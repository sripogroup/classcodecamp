2. Differentiation Tag
Tags Similarities Differences
<hr>
  • เป็น empty element
  • ใช้เป็นตัวขั้นระหว่างการเปลี่ยนหัวข้อของเนื้อหาบนเพจ
  • โดยตั้งต้นจะแสดงผลแบบ block โดยมีลักษณะเป็นเส้นแนวนอนตามความยาวของจอ
<br>
  • ใช้ในการใส่ line break เพื่อจบบรรทัดปัจจุบันและขึ้นบรรทัดใหม่ของข้อความ
  • โดยตั้งต้นจะไม่มีการแสดงผล
<ul>
  • ใช้ในการแสดงข้อมูลที่เป็นรายการ
  • ใช้ tag <li> ในการระบุสมาชิก
  • โดยตั้งต้นจะแสดงผลแบบ block
  • แสดงรายการแบบไม่แสดงลำดับ(แบบ bulleted)
<ol>
  • แสดงรายการแบบแสดงลำดับ โดยลำดับจะแสดงด้วยตัวเลขหรือตัวอักษร
<select>
  • ใช้เป็นสมาชิกของฟอร์มเพื่อรับ input จากผู้ใช้
  • ใช้ในการแสดงรายการแบบ drop-down
  • ใช้ tag <option> ในการระบุสมาชิกของรายการที่ผู้ใช้สามารถเลือกได้
<input type="checkbox">
  • ใช้ในการแสดงช่องติ้ก
  • โดยตั้งต้นจะแสดงผลเป็นกล่องสีเหลี่ยมจัตุรัสที่จะมีเครื่องหมายขีดถูกข้างในเมื่อถูกใช้งาน
  • ใช้ในการให้ผู้ใช้เลือกตัวเลือกศูนย์ตัวหรือมากกว่าจากกลุ่มของตัวเลือก
<input type="radio">
  • ใช้ในการแสดงปุ่มวิทยุ
  • โดยตั้งต้นจะแสดงผลเป็นวงกลมที่จะมีจุดข้างในเมื่อถูกใช้งาน
  • ใช้ในการให้ผู้ใช้เลือกตัวเลือกหนึ่งตัวจากกลุ่มของตัวเลือก
  • ในกลุ่มตัวเลือกแบบปุ่มวิทยุจะมีปุ่มที่ถูกเลือกได้เพิ่งหนึ่งปุ่มเท่านั้น เมื่อผู้ใช้กดเลือกปุ่มอื่นในกลุ่ม ปุ่มที่ถูกเลือกอยู่ก่อนหน้าจะไม่ถูกเลือกแล้ว
<div>
  • ใช้ในการครอบ element อื่นๆในเอกสารและใช้เป็นจุดอ้างอิง เพื่อเตรียมกลุ่มของ element เหล่านั้นสำหรับการตกแต่งด้วย CSS และการเพิ่มพฤติกรรมด้วย JavaScript
  • ใช้ในการแบ่งส่วนของเอกสาร
  • โดยตั้งต้นบราวเซอร์จะใส่ line break ให้ก่อนหน้าและข้างหลัง <div>
  • โดยตั้งต้นจะมีการแสดงผลแบบ block
<span>
  • ใช้ในการรวม inline-element ต่างๆให้อยู่เป็นกลุ่ม
  • โดยตั้งต้นจะไม่ทำให้การแสดงผลของเอกสารเปลี่ยนแปลงไป
<script>
  • ใช้ในการดัดแปลงเอกสารเพิ่มเติมจาก HTML
  • ใช้ในการนิยาม JavaScript client-side script เพื่อดัดแปลงหรือเพิ่มพฤติกรรมให้เอกสาร
  • สามารถใส่นิยามของ script ลงใน <script> ได้เลย หรืออ้างอิงไปถึงไฟล์ JavaScript ภายนอกได้โดยใช้ attribute src
  • อยู่ได้ทั้งใน <head> และ <body>
<style>
  • ใช้ในการนิยามข้อมูลสไตล์ที่จะระบุว่า element ต่างๆในเอกสารควรจะแสดงผลอย่างไร
  • ไม่สามารถอ้างอิงไปถึงข้อมูลสไตล์ภายนอกได้ ถ้าต้องการจะต้องใช้ <link> แทน
  • ควรอยู่ใน <head>

Type Usage

submit ปุ่มที่ใช้ในการยื่นข้อมูลของฟอร์มไปที่ form-handler
reset ปุ่มที่ใช้ในการรีเซ็ทค่าของฟอร์มไปเป็นค่าตั้งต้น
radio ปุ่มตัวเลือกที่ให้ผู้ใช้เลือกหนึ่งตัวเลือกจากกลุ่มของตัวเลือก
checkbox ปุ่มตัวเลือกที่ให้ผู้ใช้เลือกศูนย์ตัวเลือกหรือมากกว่าจากกลุ่มของตัวเลือก
button ปุ่มที่โดยตั้งต้นจะไม่มีพฤติกรรมใดๆ
color ใช้สำหรับ input field ที่ควรจะรับสี โดย input field นั้นอาจจะมี color picker ขึ้นมาขึ้นอยู่กับการรองรับของบราวเซอร์
date ใช้สำหรับ input field ที่ควรจะรับวันที่ โดย input field นั้นอาจจะมี date picker ขึ้นมาขึ้นอยู่กับการรองรับของบราวเซอร์
datetime-local ใช้สำหรับ input field ที่รับวันที่และเวลาโดยไม่รับ time zone โดย input field นั้นอาจจะมี date picker ขึ้นมาขึ้นอยู่กับการรองรับของบราวเซอร์
email ใช้สำหรับ input field ที่ควรจะรับ email address โดย email address นั้นอาจจะถูกตรวจสอบโดยอัตโนมัติเมื่อทำการยื่นฟอร์มขึ้นอยู่กับการรองรับของบราวเซอร์
file ส่วนรับไฟล์พร้อมปุ่ม "Browse" สำหรับการอัพโหลดไฟล์
month ให้ผู้ใช้เลือกเดือนและปี โดย input field นั้นอาจจะมี date picker ขึ้นมาขึ้นอยู่กับการรองรับของบราวเซอร์
number input field สำหรับการรับตัวเลข โดยสามารถระบุข้อจำกัดของตัวเลขที่ยอมรับได้ด้วย
range ส่วนควบคุมสำหรับการรับค่าตัวเลขโดยที่ค่าเจาะจงของตัวเลขไม่ใช่สิ่งสำคัญ(เช่น อาจจะใช้ slider เพื่อป้อนค่าตัวเลข) โดยสามารถระบุช่วงของค่าที่ยอมรับได้

time ให้ผู้ใช้เลือกเวลา(แบบไม่มี time zone) โดย input field นั้นอาจจะมี time picker ขึ้นมาขึ้นอยู่กับการรองรับของบราวเซอร์
url ใช้สำหรับ input field ที่ควรจะรับ URL address โดย URL นั้นอาจจะถูกตรวจสอบโดยอัตโนมัติเมื่อทำการยื่นฟอร์มขึ้นอยู่กับการรองรับของบราวเซอร์
week ให้ผู้ใช้เลือกสัปดาห์และปี โดย input field นั้นอาจจะมี date picker ขึ้นมาขึ้นอยู่กับการรองรับของบราวเซอร์
image ปุ่ม submit ที่สร้างจากรูปภาพ
hidden ใช้สร้าง input field ที่ไม่แสดงผลให้ผู้ใช้เห็น ช่วยให้ผู้พัฒนาเว็บสามารถแทรกข้อมูลที่ไม่ต้องการให้ผู้ใช้เห็นหรือแก้ไขไปพร้อมกับการยื่นฟอร์มได้
